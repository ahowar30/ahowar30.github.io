This assignment was focused around the use of HTML, CSS and Bootstrap to create a website about a subject of our choice. Our website is required to be hosted on Github Pages and we edit and add to our website by using the command line version of Git.

Part 1: Download Git and Setup Repository

Before beginning on the assignment proper, I had to first prepare a repository to store my content. This involved downloading Git from git-scm.com. After this, I created a Github account as well as the repository and Github Pages website that come with it. I then cloned it from Github to my computer to create my own local repository.

git clone https://github.com/ahowar30/CS460

Then I configured git on my machine to include my name and email.

git config --global user.name "Alex Howard"
git config --global user.email "ahoward17@mail.wou.edu"

And with that, I was setup and ready to work.
